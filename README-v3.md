# 🦆 烤鸭送货记账 v3 — 生产协同版

专为老大优化的终极版本：支持「订货 vs 实做」分离、今日生产单看板、微信直发小票图，真正贴合烤鸭制作与送货全流程。

> 💡 **一句话说明**：手机打开网页 → 记订单 → 自动生成生产单（含日期+星期）→ 转发给师傅 → 查账时灵活选日期+客户

---

## ✨ 新增功能亮点

- **📌 订货数 & 实做数分离**：万里一订 7 只，实做 5 只（回扣），月底仍按 7 只结算
- **🖨️ 今日生产单看板**：自动聚合今日所有客户订单，显示 `客户名 × 数量` + `今日总需制作：XX只`
- **📸 小票图一键生成**：点击「生成今日生产单」→ 下载 PNG → 微信发给师傅（含 `2026-06-12 周四`）
- **📅 查账页全面升级**：支持自定义日期范围 + 指定客户筛选 + 按客户分组显示（订货/实做/金额/未结）
- **🔤 全中文极简界面**：按钮更大、文字更直白（`未结` 替代 `赊账`）

## 📱 给老大的使用指南（微信可发）

### ✅ 记账（送鸭前）
1. 点顶部「📝 记账」
2. 输入客户名 → 填「订货数量」（如 7 只）
3. 「实做数量」默认=订货数，如回扣可手动改（如 5 只）
4. 选「未结」→ 点「✅ 记录本单」

### ✅ 查看今日生产单（发给师傅）
1. 点顶部「🖨️ 生产单」
2. 看到 `烤鸭生产单 · 2026-06-12 周四`
3. 点「📸 生成今日生产单」→ 下载 PNG → 微信发给师傅

### ✅ 查账（月底对账）
1. 点顶部「🔍 查账」
2. 选日期范围（默认当月1日→今日）
3. 选客户（或「全部客户」）→ 点「🔍 查询账单」
4. 查看每家「订货/实做/金额/未结」
